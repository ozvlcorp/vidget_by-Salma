# MoySklad — Payment Split Widget

Виджет для МойСклад: пользователь вводит общую сумму и валюту, распределяет
её между произвольным числом контрагентов (можно добавлять/убирать строки),
выбирает юр. лицо и тип документа — и виджет создаёт в МойСкладе один
документ на каждого контрагента:

- **Приходный ордер** (`cashin`)
- **Входящий платёж** (`paymentin`)

## Стек

React + TypeScript + Vite + Tailwind CSS. Обращения к МойСклад API идут через
`/api/moysklad/*`, который проксируется Vite (в разработке) или nginx (в
проде) на `https://api.moysklad.ru/api/remap/1.2` — это полностью убирает
проблему CORS в браузере.

## Разработка

```bash
npm install
npm run dev
```

Для локальной проверки без встраивания в МойСклад передайте токен через URL:

```
http://localhost:5173/?token=ВАШ_ТОКЕН
```

В продакшене МойСклад открывает виджет в iframe с `?contextKey=...&accountId=...`;
`src/App.tsx` обменивает их на токен через `VITE`-виджет-бэкенд
(`WIDGET_NAME` в `src/App.tsx` нужно выставить в соответствие с тем, как
виджет зарегистрирован на бэкенде).

## Сборка и деплой

```bash
npm run build   # tsc -b && vite build → dist/
npm run lint
```

- `Dockerfile` + `nginx.conf` — сборка образа со встроенным прокси к
  МойСклад API.
- `netlify.toml` — тот же прокси через Netlify redirects, для деплоя на
  Netlify.
